Key Factors

These are the factors that guide us on how data science is implemented in our day-to-day lives starting,
from the first step of data collection to the very last step of making meaningful insights of our data.

Data collection

This requires one to visit the field and collect data from the area of interest.
One should obtain from a trusted source so that he is able to make accurate predictions based on the data at hand.

Data cleaning

This is the step where outliers, duplicates, and missing values are dealt with so as to come up with clean data.

Data analysis

This is the step where data is analyzed with different tools and techniques to come up with
meaningful insights that show the different trends and patterns of a given dataset.

Machine learning

A model is created so that it can be used to carry out predictive analysis and visualize the data so as to foretell the future in terms of performance of a given entity.
AI can also be introduced so as to study our data and easily work with it in terms of prediction.
